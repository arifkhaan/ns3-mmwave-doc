void ns3::mmwave::MmWaveEnbPhy::EndSlot ( void  )

Marks the end of the current NR slot.

Periodically, it alternates with StartSlot in order to scan through the different NR slots. The various frame/subframe and slot counters get updated and, finally, a call to StartSlot is scheduled in order to start the next NR slot.
