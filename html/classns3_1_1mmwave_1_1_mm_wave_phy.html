◆ GetNextSlotDelay()

Time ns3::mmwave::MmWavePhy::GetNextSlotDelay ( )

The purpose of this function is to sync the slot duration to the correct value.

Since the OFDM symbols duration can not be expressed in ns as an integer, each of these symbols has a slightly lower duration (by a fraction of ns) than it should. Therefore, a sync anchor is needed in order to align the slots starting time.

Returns
the exact delay between the current TTI end and the next slot start
