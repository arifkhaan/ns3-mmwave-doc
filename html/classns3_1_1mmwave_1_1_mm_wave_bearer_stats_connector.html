Detailed Description

This class is very useful when user needs to collect statistics from PDCD and RLC.

It automatically connects MmWaveBearerStatsCalculator to appropriate trace sinks. Usually user do not use this class. All he/she needs to to do is to call: LteHelper::EnablePdcpTraces() and/or LteHelper::EnableRlcTraces().
