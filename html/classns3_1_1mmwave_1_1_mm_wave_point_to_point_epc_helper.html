Detailed Description

Create an EPC network with PointToPoint links.

This Helper will create an EPC network topology comprising of a single node that implements both the SGW and PGW functionality, and an MME node. The S1-U, S1-AP, X2-U and X2-C interfaces are realized over PointToPoint links.
